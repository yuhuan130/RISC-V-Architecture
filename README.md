### Custom CPU Architecture Implementation

Built a custom single-cycle CPU from scratch by designing and integrating the datapath and control logic for a subset of assembly instructions. Implemented core components including the program counter, register file, ALU, immediate generator, branch logic, and memory interface to enable end-to-end instruction execution.

**Key Contributions**
- Designed and implemented a complete single-cycle datapath
- Built a control unit for instruction-specific execution signals
- Added support for arithmetic/logic operations, load/store instructions, branches, and jumps
- Debugged and validated processor behavior through modular testing
- Improved circuit organization through clean component separation

**Skills Demonstrated**
- Computer architecture
- Digital logic design
- Processor datapath implementation
- Hardware control systems
- Low-level debugging and verification
- Assembly / ISA understanding
